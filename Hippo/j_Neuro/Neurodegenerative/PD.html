Criteria: MDS Clinical Diagnostic Criteria for PD

  • Parkinsonism
    • Bradykinesia
      • +
    • Rest tremor
    • Lead-pipe rigidity
  • Exclusion criteria
    • Cerebellar abnormalities
    • Downward vertical supranuclear gaze palsy / selective slowing of downward vertical saccades
    • probable bvFTD / PPA
    • Parkinsonian features restricted to the lower limb
    • dopamine receptor blocker / a dopamine-depleting agent
    • response to high-dose levodopa (-)
    • cortical sensory loss / clear limb ideomotor apraxia / progressive aphasia
    • presynaptic dopaminergic system
  • Red flags
  • Supportive criteria
    • Response to DAergic therapy
    • Levodopa-induced dykinesia
    • Rest tremor of a limb
    • Olfactory loss / Cardiac sympathetic denervation on MIBG scintigraphy
