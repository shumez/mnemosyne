109E45 母乳栄養

1ヵ月の乳児．
健康診査のため母親に連れられて来院した．母親の妊娠・分娩経過に異常はなく，母乳栄養で体重は27g/日増加している．便は黄色である．皮膚はやや黄染している．胸腹部に異常を認めない．
この児で正しいのはどれか．

  1. 便はアルカリ性を示す．
  2. 体重増加は不良である．
  3. 人工栄養児より便は硬い．
  4. 黄疸のため母乳を中止する．
  5. 腸内細菌叢としてビフィズス菌が多い．

True: e



    • 母乳栄養児の便: 酸性
    • 人工乳栄養児の便: アルカリ性
  2. 母乳栄養児: ≤ 3Mo 25-30g/d
    • 母乳栄養児: 便やわらかい
    • 人工乳栄養児; 硬め
  4. やや黄染している程度
    • 母乳栄養児: ビフィズス菌
    • <人工乳栄養児: 大腸菌, 腸球菌/li>

産褥期・初乳について正しいのはどれか．

  2. 褐色悪露から赤色悪露に変化する．
  3. 産褥熱の主な起炎菌はB群レンサ球菌〈GBS〉である．
  4. 初乳は成乳に比べてラクトアルブミンが多く含まれる．

100E8 人工乳

人工栄養と比べて母乳栄養で正しいのはどれか．

  1. 便が軟らかい．
  2. 蛋白の代謝負担が多い．
  3. アレルギーの発症が多い．
  4. 呼吸器感染症の罹患率が高い．
  5. 乳児ビタミンK欠乏症の頻度が低い．

True: a



  1. 母乳の方が, 軟便
  2. 人工乳 タンパク代謝負担が多い
  3. 人工乳 アレルギー
  4. 母乳 Ig(++)
  5. 母乳 vit.K少ない

母乳の利点

  • IgA++
  • 乳糖(++)
  • 乳児突然死症候群 SIDSが低い

母乳の欠点

  • Vit.K不足 ⇒ 頭蓋内出血
  • 薬剤移行
