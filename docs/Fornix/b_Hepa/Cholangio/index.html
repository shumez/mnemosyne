胆嚢腺筋腫症

d

胆嚢腺筋腫症
  • 胆囊粘膜が壁内に入り込んだ構造（Rokitansky-Aschoff洞）が増殖して胆囊壁の肥厚，上皮の過形成を呈した病態を胆囊腺筋腫症と呼ぶ．
  • 超音波検査では胆囊壁の肥厚や，コメット（彗星）様エコーと言われる高エコーが認められる．
  • 胆囊ポリープ同様，無症状の場合は治療を必要としないが，症状を認める場合や癌が疑われる場合は胆囊摘出術を行うことがある．
Rokitansky-Aschoff sinus
  • 胆囊の肥厚した筋層に粘膜から通ずる小さな洞を形成したものである．
    慢性胆囊炎，胆囊腺筋腫症などでよくみられる．
  • 胆囊造影で，胆囊の輪郭に沿う点状陽性像としてみることができる．
