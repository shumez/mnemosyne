88B27 チアノーゼ

チアノーゼ性先天性心疾患でみられるのはどれか．3つ選べ．

  1. 高血圧
  2. 多血症
  3. 脳血栓
  4. 脳膿瘍
  5. 心筋梗塞
  • 多血症 ⇒ 脳血栓
  • 奇異性塞栓 ⇒ 脳膿瘍
  • R→L 乱流 ⇒ 心内膜炎

b c d

VSD
軽症 軽症では，汎収縮期雑音か収縮後期に増強する収縮期心雑音を聴取する．雑音は強大で胸壁にthrillを触れ，Roger雑音と呼ばれる．Ⅱ音は正常で（＝肺高血圧はない），チアノーゼもない
中等症 肺血流が増加すると，左房拡大に伴い拡張期ランブルを認める（相対的僧帽弁狭窄＝Carey Coombs雑音）．Ⅱ音は次第に亢進し，収縮期雑音はダイアモンド型となる．この時点では，まだチアノーゼはない．
Eisenmenger 過度の肺血流増加を防止するため，生体では自らの肺動脈の内中膜を肥厚させる．肺高血圧は進行し，Ⅱ音はさらに増高する．最終的には肺血流量は減少し，左房拡大はなくなり拡張期ランブルも消失し，心室中隔欠損を介して右左短絡となる（＝チアノーゼの出現）．この時点で肺動脈弁閉鎖不全の拡張期雑音を聴取する（Graham Steel雑音）．相対的な肺動脈弁狭窄によるわずかな収縮期雑音を聴取する場合もあるが，全く雑音を聴取しない場合も多い．
