    立位による腎血流低下でRAA系を刺激し，レニン活性およびアルドステロンが反応するか否かをみる検査．ループ利尿薬（フロセミド）を併用することもある（立位−フロセミド負荷試験）．RAA系には依存しない原発性アルドステロン症ではレニンは抑制されたままでアルドステロンも無反応，RAA系に依存する過形成ではアルドステロンの分泌が増加するとされるが，例外もある．

    強力な合成グルココルチコイドであるデキサメサゾンを前日23時に投与し，翌朝のコルチゾールを測定する．正常であればコルチゾールは抑制され，ACTH−コルチゾール系に自律的機能亢進があれば抑制されないか，抑制が不十分である．
