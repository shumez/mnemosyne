ツツガムシ病

d

ツツガムシ病
  • 概念
    O.tsutsugamushiに感染したツツガムシ（ダニの一種）に刺咬されて発症する．高熱，発疹，特有の刺し口が特徴である．わが国では昭和55年以降，年間数百人が発症している．
  • 診断
    ①野外活動後に発症（潜伏期5～14日），
    ②悪寒，発熱，頭痛，体幹から広がる全身の麻疹様皮疹，
    ③胸腹部・腋窩・陰部等の虫による刺し口（有痛性紅斑），
    ④所属リンパ節の有痛性の腫大，
    ⑤WBC↓，血小板↓がみられる．
    ➡ツツガムシ病を考える． ●確定診断は血清学的診断で抗オリエンチア・ツツガムシ抗体（IgMの検出，IgGの有意上昇），またはPCR法で血液・刺し口（痂皮）からのオリエンチアDNA検出による．
  • 治療
    治療が遅れると，DICを合併し，主な死因となる．
    ●テトラサイクリン系（第一選択薬），クロラムフェニコールなどの投与．
    ※β-ラクタム系薬は一切無効であり，マクロライド系，ニューキノロン系も効果がない．
比較的徐脈
    • 腸チフス・パラチフス
    • クラミジア肺炎
    • ツツガムシ病
