肺クリプトコッカス
  1. 胸部単純X線写真，胸部CT写真
    • 単発性もしくは多発性の結節影が認められる．
    • 空洞，spiculaや胸膜陥凹などの多彩な陰影を呈することがあり，画像のみでは肺癌，肺結核との鑑別が難しい．
  2. 血清診断
    • グルクロノキシロマンナン抗原（莢膜多糖抗原）陽性　※β-Dグルカンは陰性
  3. 真菌学的検査
    • 喀痰やBALFを検体とした塗抹標本（PAS染色やGrocott染色など）や培養，TBLBで得られた組織でのクリプトコックスの証明
    • ※病理学的検査では，組織反応として肉芽腫形成がみられる．
  4. その他
    • 肺クリプトコックス症と診断し，中枢神経症状を有する場合は，脳脊髄液を検査し，中枢神経病変の有無を確認する．
    • 重症例では，血液培養が陽性となる場合もある．
