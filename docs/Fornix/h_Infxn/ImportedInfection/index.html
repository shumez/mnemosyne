ランブル鞭毛虫

d

ランブル鞭毛虫
流行地 世界中に分布するが,特に熱帯や亜熱帯の発展途上地域では多い.
感染経路 食事や飲水時に囊子型を経口摂取して感染する．先進国でも水道水を介して集団感染した事例がある．男性同性愛者の間でヒト−ヒト感染することもある．
潜伏期間 1〜2週間程度.
症状 下痢（軟便から水様便までその程度は様々である．また脂肪便をみることもある）が長期間続く．まれに胆管炎，胆囊炎を起こすことがある．
診断 新鮮な便で，顕微鏡下にランブル鞭毛虫の栄養型や囊子型を検出する．
治療 メトロニダゾールの経口投与．
