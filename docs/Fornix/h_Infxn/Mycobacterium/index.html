粟粒結核
  • 概念
    多量の結核菌が血流内に入って起こる重篤な血行性播種性結核症で，少なくとも2つ以上の全身の臓器に粟粒大あるいはそれに近い結節性散布巣をつくる．約30％は髄膜炎を発症する．肝疾患，血液疾患，悪性腫瘍，糖尿病，ステロイド，免疫抑制薬，血液透析，AIDSなどの免疫不全状態が誘因となる．成人よりも抵抗力の弱い小児の方が罹患しやすい．
  • 診断
    ①好発：免疫不全状態の患者
    ②悪寒を伴う稽留熱，全身倦怠感がみられる．
    ③赤沈↑，CRP（＋），WBC↑，〈炎症所見＋類白血病反応〉
    ④leukoerythroblastosis（末血に赤芽球が出現），ツ反陰性化を伴う．
    ⑤胸部X線で，両側肺野にびまん性小結節～斑状影がみられる．
    ➡粟粒結核を考える．
    ●確定診断は，痰，血液，尿培養などからの結核菌の検出によるが検出率は低い．菌が検出されないときは，眼底検査で肉芽結節を認めるか，肝生検，骨髄生検，経気管支肺生検（TBLB）で結核の病理組織（乾酪性肉芽腫）の有無を確認する．
  • 治療
    INH，RFP，SM（or EB）を中心とした強力な抗結核療法．

TB感染の特徴
  • ①感染と発症は別：
    結核に感染後，発症する例は10％程度といわれており，残りは潜伏感染となる．半数が2年以内に発症するが，発症まで数十年かかる例もある．結核に対する免疫反応獲得の証明（ツベルクリン反応・インターフェロンγ試験陽性）は感染を示すが，発症を示すものではない．
  • ②感染経路とその遮断：
    結核は空気感染形式（飛沫核型）であり，感染経路の遮断のために患者にサージカルマスクを着用させ，医療スタッフはN95などの微粒子除去マスクを着用する．その上で個室隔離（可能であれば陰圧室）を行う．
  • ③感染性がある病型：
    肺結核や咽頭結核である．一般に肺外結核では感染性がないが，結核性胸膜炎などで肺内に微小な病変を生じた場合は感染性を持つことがある．
  • ④感染性を高める因子：
    喀痰中の抗酸菌塗抹陽性，空洞病変がある場合，患者が高い社交性を持つ場合，患者の職業（芸人，俳優，歌手など）
  • ⑤感染性を高める状況：
    咳を誘発するような気道内吸引操作，気管内挿入，気管支内視鏡検査，狭い空間（内視鏡室，カラオケボックスなど）
TB 感染拡大予防
  • ①接触者検診：
    肺結核患者と知らずに曝露した家族や職場の同僚，学校の同級生，医療従事者などに対して行う検診を「接触者検診」という．これは発症者からの二次感染，三次感染の連鎖を断ち切ることを目的としている．
  • ②ハイリスク症例：
    結核が感染しやすく重症化しやすい乳幼児，免疫抑制状態の症例などは接触者検診の中で「ハイリスク症例」と位置づけられる．
  • ③danger group：
    結核感染を受けて発症（二次感染）すると，さらに多くのハイリスク症例へ感染させる（三次感染）可能性がある保育士，教師，医療従事者など．
