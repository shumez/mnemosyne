特発性器質化肺炎
  • 病態
    • 肺胞腔内にポリープ状器質化病変を形成する間質性肺炎．
    • 亜急性に呼吸困難，咳嗽，発熱を認め，細菌性肺炎と間違われることも多い．
  • 好発
    • 50代．性差はない．
  • 治療
    • ステロイドが有効で予後は良好である．
