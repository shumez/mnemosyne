  • 概念
    熱により生体の組織が損傷されたものを言い，その重症度は，受傷面積・深達度・部位により診断する．初期救急が非常に重要で，意識障害がある場合や気道熱傷を疑うときにはまず気管内挿管を行う必要があり，また熱傷による血管透過性の亢進，水・蛋白の漏出により熱傷ショックに陥ることがあるため，直ちに太い輸液ルートを確保する必要がある．
  • 診断
    • ①誘因：ガス爆発，過熱した風呂などでの受傷後
    • ②皮膚の発赤（＋），疼痛（＋），水疱（－）であるとき，
      ⟹ Ⅰ度熱傷と診断する．
    • ③皮膚に水疱（＋），疼痛（++）であるとき，
      ⟹ Ⅱ度熱傷と診断する．
    • ④皮膚の蒼白・炭化，疼痛（－）であるとき，
      ⟹ Ⅲ度熱傷と診断する．
  • 治療〕熱傷の面積・深度・部位により，重症度を判断する．
    1. 気道熱傷が疑われれば，まず気管挿管
    2. 広範囲熱傷では，Baxter法に基づくリンゲル液の輸液
      ⟹ Baxter法：リンゲル液4mL×熱傷面積（％）×体重（kg）
    3. 浮腫による血行障害があれば，局所的減張切開．
