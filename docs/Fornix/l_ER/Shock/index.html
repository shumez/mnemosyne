神経原性ショック
  • 概念
    • 迷走神経，交感神経系などの循環調節を担う神経の障害により，末梢血管抵抗の減少した血圧の低い状態を呈したものである．徐脈が特徴．
  • 診断
    • ①患者にストレス，疲労，不安，疼痛や，　〈迷走神経反射〉
    • ②高位脊髄の外傷，血管障害，腫瘍，　〈脊髄横断障害〉
    • ③脊椎麻酔や脊髄損傷，脳幹部の障害などの背景がある．
    • ④血圧↓，徐脈，CVP↓，末梢血管抵抗↓，冷汗などがみられる．
    ⟹ 神経原性ショックと診断する．

    脈拍↓，血圧↓には，まず硫酸アトロピン

    • ②輸液
