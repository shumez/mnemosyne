    例：眠剤中毒などに続発した気管支肺炎，外傷に続発した破傷風
  • ③ ①または②の疑いがあるもの
  • ④診療行為に関連した予期しない死亡及びその疑いのあるもの 例：注射・麻酔・手術・検査・分娩など，診療行為中，または比較的直後における予期しない死亡
