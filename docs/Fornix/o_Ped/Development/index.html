表在反射 皮膚，粘膜受容器への刺激により生ずる反射性筋収縮．角膜反射，咽頭反射，睾挙筋反射，手掌反射，腹壁表在（腹皮）反射，足底反射など．

姿勢反射 位置覚に関与する全身の受容器によって，姿勢や運動時の平衡を保つために働く一群の反射．パラシュート反射，緊張性頸反射，Landau反射など．

交叉性伸展反射 0〜2ヵ月 仰臥位にし，一方の膝を押さえて伸展させ，同側の足底を刺激すると，反対の下肢を屈曲してから検者の手を払いのける．
Moro反射 0〜3-4ヵ月 仰臥位にして，頭部を30度持ち上げ，急に手の平の上に落下させると，両側上肢を伸展・外転して手を開く．
緊張頸反射 2〜6ヵ月 仰臥位にした新生児の顔を左右どちらかに向けると，向けたほうの上下肢が伸展，対側が屈曲する．
追っかけ反射/吸啜反射 0〜4-7ヵ月 口角や頬に指を触れると，それを追いかけ口に含もうとする/口の中に指を入れると規則的に吸う行動をとる．
手掌把握反射 0〜6ヵ月 児の小指側から検者の指を入れ手掌を圧迫すると握り返してくる．
足底反射 0〜10ヵ月 仰臥位にし，母趾球を検者の母指で圧迫すると全趾が屈曲する．
Babinski反射 0〜約24ヵ月 足底外側を擦過すると，母趾を開扇させ，背屈が生じる．
Landau反射 3〜24ヵ月 腹臥位にして胸部を持って空中に抱え，頭部を背屈させると脊柱と下肢が伸展し，頭部を前屈させると脊柱，下肢が屈曲する．
パラシュート反射 9ヵ月〜一生 立位に抱えて急に頭部を床に近づけると両手を伸ばして支えようとする．
