急性硬膜下血腫の病態

乳児揺さぶられ症候群 SBS Shaken Baby Syndrome

  • 定義： ①眼底出血，②硬膜下出血あるいはくも膜下出血，③頭部に外表創，骨折などの直達外力が加わった所見をほとんど認めない，の3点を特徴とする乳幼児の症候群．被虐待児症候群における頭部外傷の受傷機転の一つ．以前は乳幼児鞭打ち様症候群（shaken baby whiplash syndrome）と呼ばれていた．
  • 原因： 乳幼児の上半身が揺さぶられることが多い被虐待児に好発するため，「両肩を掴んで体を前後に揺する（一種の恫喝）」という行為が，頭部の鞭打ち様の動きを起こし架橋静脈の断裂によって生じると考えられていた．しかし被虐待児のほとんどは頭部に外表創や頭蓋骨骨折を伴っているので，頭部の打撃が主な原因という報告も多い．
  • 好発年齢： 乳幼児．①頭部が相対的に重い，②支える首もすわらず，定頸後も筋力が弱い，③大泉門や骨縫合が存在し弾力に富む，④神経線維を取り巻く髄鞘の発達が未熟なため脳が柔らかい，⑤側脳室，くも膜下腔が相対的に大きい，などの乳幼児の特徴から，揺さぶりによって脳に加速・減速力，回転力が作用すると，脳と頭蓋を結ぶ血管が断裂し，硬膜下出血やくも膜下出血が生じやすい．同様の機序により脳神経や頸髄の神経も損傷を受けることがある．
  • 眼科的所見： 揺さぶりによって水晶体とともに硝子体も動揺する結果，硝子体・網膜間の血管が断裂して網膜出血をきたす（75～90％）．硬膜下出血が両側の場合や網膜出血が存在する場合は揺さぶりがあったことを示す証拠になる．
