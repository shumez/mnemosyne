子宮破裂
  • ①定義：
    主として分娩時に起こる子宮体部ないしは子宮下部の裂傷をいう．
  • ②分類：
    子宮破裂には，完全子宮破裂と不全子宮破裂がある．完全子宮破裂は子宮漿膜面を含む子宮壁の全層の裂傷ないしは破裂であるが，不全子宮破裂は，子宮漿膜面には裂傷が及ばない．
  • ③原因：
    瘢痕破裂，外傷性破裂，自然破裂の3つに分類される．
    • 瘢痕破裂は，帝王切開や筋腫核出術，子宮内搔爬といった手術による子宮筋層の抵抗減弱などにより起こる．
    • 外傷性破裂は，交通事故などの外傷，骨盤位牽出術，回転術，鉗子分娩，巨大児分娩，回旋異常分娩，陣痛誘発剤乱用などによって起こる．
    • 自然破裂の原因は，子宮下部の過度伸展によるもので，狭骨盤，骨盤内の腫瘍，巨大児分娩，回旋異常や胎位異常分娩などがある．
