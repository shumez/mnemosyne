104A3 標準治療

  2. 温熱療法
  3. 放射線治療
  4. 抗癌化学療法
  5. ホルモン療法

b

  • 1．局所療法　局所領域に再発した乳癌に対しても適応あり
    • 手術…乳房部分切除あるいは乳房切除（全摘）術
    • 放射線療法…乳房温存手術後の全乳房に対する照射が一般的．再発性の高い乳房切除術例に対する鎖骨上や胸壁への術後照射，（鎖骨上・内胸などのリンパ節）転移症例や骨転移症例に対する疼痛緩和目的にも用いられる．
  • 2．全身療法　術前，術後の補助療法，転移性乳癌の延命目的
    • ホルモン療法…ホルモン受容体陽性乳癌に対して（乳癌の6～7割はホルモン依存性）
    • 抗癌化学療法（分子標的薬を含む）
  • ※実際の治療法の選択に際しては，病期，ホルモン依存性，HER2遺伝子の状態などより総合的に判断する．
