  2. 縮瞳薬注入
  3. 水晶体前囊切開
  4. 水晶体乳化吸引
  5. 眼内レンズ挿入

e

白内障手術
  • ①角膜または強角膜部切開
  • ②水晶体前囊切開
  • ③水晶体囊-皮質の分離（水晶体核，皮質の吸引除去操作を容易にする目的）
  • ④水晶体核の破砕（乳化）と吸引除去
  • ⑤水晶体囊内への眼内レンズ挿入
  • ⑥眼内レンズの位置の修正
