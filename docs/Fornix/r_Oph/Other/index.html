102G43　原因

25歳の男性．
流涙と眼瞼腫脹とを主訴に救急外来を受診した．
早朝から海釣りに行き，一日中，晴天下の洋上にいた．帰宅して午後11時を過ぎたころから両眼の痛みが現われ次第に増強してきた．流涙，両眼眼瞼腫脹および結膜の充血があり，眼を開けられなくなった．細隙灯顕微鏡検査で両眼に点状表層角膜症がみられる．
原因で最も考えられるのはどれか．

  1. 異物
  2. 細菌
  3. 脱水
  4. 赤外線
  5. 紫外線
