耳下腺腫瘍 (s/o 多形腺腫)

d

  1. 側頸嚢胞: 胸鎖乳突筋と頸動静脈の間
耳下腺腫瘍
  • 病理
    • 上皮性良性腫瘍が多く，特に多形腺腫（混合腫瘍）が最も多い．
      • 浸潤性に発育しないので，表面は平滑．
      • 顔面神経麻痺をきたすことはまずない．（悪性腫瘍との鑑別）
      • その他，上皮性では，乳頭状囊腺腫（Warthin腫瘍：耳下腺腫瘍の15％程度で，高年男性にみられる），oncocytomaがある．
    • 非上皮性のものでは血管腫（小児の耳下腺腫瘍で最多），リンパ管腫，神経原性腫瘍などが発生する．
    • 悪性腫瘍では，粘表皮腫，腺様囊胞癌，腺房細胞癌，扁平上皮癌などがある．
  • 症状
    • 無痛性腫瘤の形成が主である．
    • 悪性のものでは，疼痛，皮膚への浸潤，顔面神経麻痺，転移形成をみる．
  • 治療
    • 耳下腺腫瘍摘出．悪性の場合には放射線療法，化学療法を併用．
