アレルギー性鼻炎

d

  4. allergy
アレルギー性鼻炎
  • 〔概念〕
    • アレルギー性鼻炎は，鼻粘膜のIgE依存性のⅠ型アレルギーで起こり，副交感神経機能亢進状態により，鼻汁・くしゃみ・鼻閉をきたす．
  • 〔原因〕
    • 吸入性アレルゲン：ハウスダスト（ダニ），花粉（スギ，ヒノキなど）の2つが重要．
    • 接触性アレルゲン：化粧品，点鼻薬など
  • 〔症状〕trias−
    • ①発作性くしゃみ
    • ②水様性鼻汁
    • ③鼻閉
    他に，鼻咽頭・眼などの瘙痒感，全身倦怠感を伴う．
  • 〔合併症〕
    • 喘息（ハウスダストで多），結膜炎（スギ花粉で多）などの他のアトピー性疾患を合併するケースも少なくない．※小児では特にハウスダストが重要． ※くしゃみ発作は，三叉神経終末からの反射によって起こる．
  • 〔検査〕
    • 鼻汁中の好酸球の証明は診断に有用である． 鼻腔粘膜の鼻鏡検査では，くしゃみ発作時に下鼻甲介は蒼白で腫脹する．
