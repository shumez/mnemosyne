椎体骨折の分類
  • 圧迫骨折：
    屈曲外力による前方支柱の損傷．安定型である．脊髄損傷の合併はまれ．
  • 破裂骨折：
    垂直圧迫力による前方および中央支柱の損傷．骨片が脊柱管内に転位し，しばしば脊髄損傷を合併する．
  • 脱臼骨折：
    伸展力，屈曲力，回旋力などが重複して発生する．three-column（脊柱の前方・中央・後方）すべての損傷．不安定型骨折である．脊髄損傷をほぼ合併する．
骨粗鬆症を有する人に起こりやすい骨折
  • 脊椎圧迫骨折，大腿骨転子部骨折，大腿骨頸部骨折，上腕骨外科頸骨折，Colles骨折（橈骨遠位端）
