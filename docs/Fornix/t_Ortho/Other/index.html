d

胸郭出口症候群の主な症状
  • 腕神経叢圧迫症状：
    上肢挙上により上肢の疼痛・しびれ・だるさの増悪および再現．
  • Morley テスト陽性，脈管圧迫テスト陽性．
  • 腕神経叢牽引症状：
    上肢下垂時に症状が強く，上肢下方牽引により増悪．
    上肢・肩甲帯の持ち上げによる症状改善．
胸郭出口症候群の主な診断方法
  • Morley test:
    鎖骨上窩斜角筋の圧迫と上肢の放散痛
  • Wright test:
    両肩外旋外転位で橈骨動脈の拍動が止まる
胸郭出口症候群の治療法
  • 保存療法：しびれの原因となる作業の中止 ⟹最も根本的．
  • 手術療法：
    重症例では，例えばWright テストの肢位で鎖骨下動脈の造影を行い狭窄が示されれば，第一肋骨の切除を行ったりする．前斜角筋切除術は腕神経叢の症状の強いものに対して行う．
