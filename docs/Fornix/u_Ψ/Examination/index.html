110B1 心理・精神機能検査

決められた10枚の図版を順番に提示して施行する心理・精神機能検査はどれか．

  1. Rorschachテスト
  2. Minnesota多面人格検査〈MMPI〉
  3. Wechsler成人知能検査〈WAIS-Ⅲ〉
  4. Mini-Mental State Examination〈MMSE〉
  5. ウィスコンシンカードソーティングテスト〈WCST〉

a

心理学的検査
Minnesota多面人格検査（MMPI） 質問紙法による人格検査の代表的なもの．
550の質問に対しyes，noで答えさせ，10の臨床尺度および4の妥当性尺度により被験者の人格プロフィールを表す．
Rorschachテスト 投影法による人格検査の代表的なもの．
インクのしみでできた10枚の図版を被験者に示して，何に見えるかを答えさせ（図版に自己を投影させて），その応答を分析することで人格面を検査する．
簡易精神症状評価尺度（BPRS） 記載された事項について質問を行い（構成的面接），18症状の項目をそれぞれ7段階に評価するもの（医師が評価する）．すべての精神病，特に統合失調症を対象とする．
ハミルトンうつ病評価尺度（HRS-D） 検者が評価する他覚的なうつ病の症状評価尺度である．17項目の症状の重症度（3〜5段階）の評価を行い，16点以上をおおむね中等度のうつ病とする．後に項目が追加されて21項目となった．
ベック〈Beck〉のうつ病自己評価尺度 うつ病に関する21項目に関して，それぞれ4段階で自己評価する．
Zungのうつ病自己評価尺度 うつ病に関する自己記入式の評価尺度で，患者自身が評価する．
知能検査
Wechsler成人知能検査（WAIS） 16歳以上の成人に用いられる知能テスト．言語性検査（6項目）と動作性検査（5項目）から，それぞれについての実際の得点とその年齢に対して期待される平均得点の比から知能指数（IQ）を算出する．
Wechsler児童用知能検査（WISCR） WAISの小児版．対象年齢は5〜15歳．
Mini-Mental State Examination（MMSE） 認知症の評価法である．見当識，記銘・想起，注意と計算，言語，視覚構成の各項目があり，30点満点のうち，23点以下で障害の疑いを示す．
改訂長谷川式簡易知能評価スケール（HDS-R） 認知症の評価法である．
見当識，記銘・想起，計算，言語の流暢性の各問があり，総合合計点は30点，20点以下を認知症の疑いと判定する．
対語記銘力検査 有関係対語（煙草−マッチなど）と無関係対語（少量−畳など）を被験者に記銘させ，あとで想起させる検査である．
記銘力テストの一種である．
神経心理学的検査
標準失語症検査（SLTA） 失語症のテストである．患者の言語症状を5つの大項目（Ⅰ聴く，Ⅱ話す，Ⅲ読む，Ⅳ書く，Ⅴ計算）と26の下位項目に分け，6段階の評価尺度で評価する．

    • BPRSの18項目を含む30項目からなり，陽性症状，陰性症状，総合精神病理の評価を含む．

    • 曖昧な刺激に対する反応により被験者のパーソナリティのさまざまな面を検討する．心理的負担があるため病状が落ち着いている時期に行う．
  • ウィスコンシンカードソーティングテスト（WCST）
    • 色・形・数がそれぞれ4種ずつあるカードを被験者に分類させることにより，作業記憶（ワーキングメモリ）に関連する前頭葉機能を調べる検査．遂行機能障害の評価に有用である．
  • 統合失調症認知機能簡易評価尺度（BACS）
    • 統合失調症による注意力，記憶力，遂行機能などの認知機能障害を簡便に評価可能．
  • 薬原性錐体外路症状評価尺度（DIEPSS）
    • 抗精神病薬を使用中の患者の錐体外路症状の重症度を評価する．
