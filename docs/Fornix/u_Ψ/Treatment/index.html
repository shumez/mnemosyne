  2. 心気症 － 遊戯療法
  3. 強迫神経症 － 行動療法
  4. ヒステリー － 精神分析療法
  5. うつ病 － 催眠療法

精神療法
治療法 方法 適応疾患
精神分析療法 無意識のうちに抑圧されていた葛藤を意識化させ，洞察し解決に向かわせる．
  • 神経症性障害など．
  • 統合失調症は適応がない
自律訓練法 自己催眠により全身をリラックスさせ，心身のコントロールを行う．治療者への依存は起こらない．
  • 心因性疾患（神経症性障害，心身症など）
  • 身体疾患による二次的不安・緊張など
森田療法 症状（苦痛）を受け入れながら目的・行動本意の作業を繰り返すことにより，症状に耐えて生活できるようにする．
  • 強迫性障害など
行動療法 学習によって獲得された誤った習慣によるものを除去し，適応行動を伸ばす．
  • 神経症性障害
  • 心身症
  • アルコール・薬物依存
  • 人格障害
  • 自閉症
系統的脱感作：低刺激の状態から徐々に慣らしていき，症状を軽快させる．
  • 摂食障害
  • 神経症性障害
オペラント技法：正の状態を示すことで，負の状態を認知させ，常に正の状態を保てるようにする．
  • 小児自閉症
認知療法 自動思考を明らかにし，その誤りを修正することによって症状の軽快を図る．
  • うつ病
心理教育
（サイコエデュケーション）		 患者やその家族に対して，疾患の正しい知識と対処を教え，精神的負担の軽減や回復の助けとする．
  • 統合失調症
集団精神療法 他者との交流を介して心身的接触を促進させ，次の社会復帰活動を準備する．心理劇，断酒会などがある．
  • 統合失調症
  • 薬物・アルコール依存症
遊戯療法 遊びを介した交流，不安や敵意などの感情の発散や表現を診断・治療手段とする．
  • 児童・認知症患者などの言語能力が不十分な患者（不登校など）
箱庭療法 患者に箱庭を作らせながら，作品に対して患者の非言語的表現を解釈し，治療に役立てる．
家族療法 夫婦関係や，小児期・思春期の患者の問題が家族関係の歪みから生じている場合に行われる．
  • 神経病性障害
  • 摂食障害
  • 人格障害
催眠療法 暗示によって症状を除去したり，葛藤を明らかにすることによって洞察を得る．
  • 神経症性障害
  • 心身症
