画像検査のまとめ
検査 X線 CT MRI エコー
原理 人体を透過したX線がフィルムやイメージングプレート（IP）に感光し，画像が得られる． 臥位の人体に360°方向からX線を照射し，X線吸収度を元にコンピュータが再構成した断層画像を得る． 磁気を利用してプロトンの緩和現象時に得られるエネルギーを信号として受け取り，任意断面の断層画像を得る． 体表から超音波を当て，反射波を観測するまでの時間によって，体表からの距離を算出し，画像化する．
放射線被曝 少ない（透視では多い） 多い なし なし
造影剤 ヨード（血管），バリウム（消化管） ヨード，気体（消化管） ガドリニウム，鉄剤 マイクロバブル
長所 安価であり，骨疾患（骨折など）や肺疾患の診断に有用．空間分解能が高い． 骨・石灰化や急性期の出血の検出，内臓の観察に有用．短時間で全身の検査を行うことができる． コントラスト分解能が高い．DWIで急性期脳梗塞の診断が可能．骨に囲まれた脳底などの病変に適する． 検査する場所を選ばない．リアルタイムで表示ができる．非侵襲的である．容易かつ連続的に血流を評価できる．
短所 内臓の観察には適さない（コントラスト分解能が低い）． MRIと比較して，病変と正常組織のコントラストがつきにくい． 検査時間が長い．体動の影響を受けやすい（小児や認知症患者に注意）． 得られる画像が，検者の経験と技術に左右される．骨や空気（肺，消化管）を描出できない．空間分解能が低い．
