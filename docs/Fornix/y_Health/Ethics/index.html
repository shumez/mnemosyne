110H1 介入を伴う臨床研究

介入を伴う臨床研究について正しいのはどれか．

  1. 倫理審査委員会の審査は必要ない．
  2. 被験者は研究計画書を閲覧できる．
  3. 開始した研究は中止することはできない．
  4. 研究に伴う有害な事象は開示しなくてもよい．
  5. 研究で得られたデータの部分的改ざんは許容されている．

b

GCPが要求する治験実施体制
  • ①治験依頼者は治験実施計画書を作成し，医療機関の長へ提出すること
  • ②原則として治験実施医療機関毎に治験責任医師を置くこと
  • ③治験審査委員会（IRB）を置くこと
  • ④治験実施医療機関と治験担当者は，治験データの信頼性確認のためのモニタリング，監査に協力すること
  • ⑤被験者への説明，同意は文書で行うこと（参加取りやめはいつでも可能）
