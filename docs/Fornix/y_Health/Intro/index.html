101D28 行動変容のステージ

63歳の男性．
3ヵ月前の健康診査で空腹時血糖が146mg/dL，HbA1c 8.0％（基準4.3～5.8）を指摘され通院中である．初診時は身長172cm，体重77kg．脈拍80/分，整．血圧136/80mmHg．食事療法と運動療法とを含む生活指導を行い，2ヵ月間で体重が2kg減少した．1週前の検査で空腹時血糖130mg/dL，HbA1c 7.5％であった．医師と患者との会話を以下に示す．

医師「こんにちは．調子はいかがですか」
患者「体重が2kgはすぐに減ったのですけれども，それからほとんど減らないんですよ」
医師「そうですね．減量を続けるのは大変ですよね．でも，血糖は130mg/dL，HbA1cも7.5％と良い方向に向かっていますよ」
患者「そうですか．でも，むしょうに甘いものが欲しくなる時があるのですよ．時々，がまんできずに食べてしまうこともありますしね」
医師「糖尿病がよくなるまで一緒に頑張りましょうよ」

行動変容のステージはどれか．

  1. 無関心期
  2. 関心期
  3. 実行期
  4. 維持期
  5. 再発期

c

  1. 減量意思+
  2. 実行している
  4. 目標到達していない, 6mo以上たっていない
  • ①無関心期（6ヵ月以内に減量や禁煙などの行動変容に取り組む意思がない）
  • ②関心期（6ヵ月以内には行動変容する意思がある）
  • ③準備期（1ヵ月以内に行動変容するつもり）
  • ④実行期（行動変容し始めて6ヵ月以内）
  • ⑤維持期（行動変容を続けて6ヵ月以上）
110H17 行動変容 Qs

行動変容について正しいのはどれか．

  1. モデリングが有効である．
  2. 行動を変えて1年間は実行期である．
  3. 準備期の次のステージは関心期である．
  4. 強化マネジメントは無関心期に重要である．
  5. 環境変化に応じて反射的に獲得するものである．

a

  2. 6mo未満
  3. 準備 → 実行
  4. 実行期, 維持期: 強化マネジメント
  5. 学習して
行動変容 stage model
無関心期
（前熟考期）
  • 行動変容を全く考えていない時期
  • 知識・情報の提供，問題点の指摘を行う
関心期
（熟考期）
  • 行動変容の意義は理解しつつもまだ行動変容がない時期
  • 動機付けを行う
準備期
  • すぐ行動を始めようとしているか，患者なりの行動変化がある時期
  • 具体的な行動提案，目標設定など計画支援を行う
実行期
  • 健康行動を始め，努力して実行している時期
  • 行動実践の意欲強化と報酬付け，環境調整を行う
維持期
  • 実行中だが，くじけそうになったり逆戻りしそうになりがちな時期
  • 家族の協力を得て，遂行へのサポートと賞賛，実行のための障害について問題解決を行う
