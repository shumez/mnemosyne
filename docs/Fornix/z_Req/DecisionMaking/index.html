105F2 エビデンス

薬の効能を検証する研究方法で最もエビデンスレベルが高いのはどれか．

  1. 無作為化比較対照試験
  2. 症例対照研究
  3. コホート研究
  4. 横断研究
  5. 症例研究

a

  • Ⅰ　システマティック・レビュー/無作為化比較対照試験のメタアナリシス
  • Ⅱ　1つ以上の無作為化比較対照試験
  • Ⅲ　非無作為化比較対照試験
  • Ⅳa　分析疫学的研究（コホート研究）
  • Ⅳb　分析疫学的研究（症例対照研究，横断研究）
  • Ⅴ　記述研究（症例研究やケース・シリーズ）
  • Ⅵ　患者データに基づかない，専門委員会や専門家個人の意見
研究デザイン
  • 1．無作為比較対照試験（ランダム化比較研究）＜RCT；Randomized Controlled Trial＞
    ある集団を無作為に分割し（治療群，非治療群など），一定期間追跡した後にその効果を検討する．最もバイアスの影響が少なく，介入（治療）の効果をみるには最適．
  • 2．コホート研究＜Cohort Study＞
    ある集団を一定期間追跡し，介入（治療）要因の有無により，疾患の発生率が異なるかどうかを観察する．予後や因果関係を「自発的に」調べるには有効（強制的に分割するわけではない）．
  • 3．患者対照研究＜Case-Control Study＞
    有害な結果が生じた患者（Case）と生じなかった患者（Control）を選択し，過去の治療を遡って追跡する．間接的にだが，治療と結果の関連（オッズ比）を推定することができる．
  • 4．横断研究＜Cross Sectional Study＞
    アンケート調査，患者調査などを期間を定めて行い，諸因子の分布を調べる．その時期における有病率などを知るためには有効だが，因果関係などを証明するのは困難．
  • 5．専門家の症例報告＜Case Report＞
    特異な症状，治療経過などを示した一例報告．応用研究への示唆を得るためには重要．
