110H19 全身麻酔 急速導入

成人に対する全身麻酔の急速導入で最初に投与するのはどれか．

  1. 純酸素
  2. 筋弛緩薬
  3. 吸入麻酔薬
  4. 静脈麻酔薬
  5. 副交感神経刺激薬
  • 麻酔導入
    1. 酸素
    2. 麻酔
    3. マスク換気
    4. 気管内挿管

a

麻酔導入
  • 急速導入（rapid induction）：
    1. 酸素化
    2. 短時間作用型の静脈麻酔薬
      • プロポフォール
      • チオペンタール
      オピオイド
      • フェンタニル
      • レミフェンタニル
      を静注
    3. 筋弛緩薬（ロクロニウム）
    • 呼吸が停止するため，マスクによる陽圧換気を行い，十分な筋弛緩が得られた状態で気管内挿管を行う．
  • 緩徐導入（slow induction）：
    • 静脈確保が困難な小児などでは静脈麻酔薬を用いることができないため，吸入麻酔（セボフルラン，笑気）のみで導入する．
    • 麻酔深度が安定して，不動かつ末梢静脈が拡張した状態で後から静脈確保し，必要な麻酔薬を投与して気管内挿管を行う．
  • 迅速導入（rapid sequence induction/crash induction）：
    • フルストマックなど誤嚥のリスクが高い患者で適応となる．静脈麻酔薬と筋弛緩薬の投与後，速やかな気管内挿管により気道を確保する．
    • 陽圧換気で胃内にガスが入り込むと嘔吐の危険性があるため，陽圧換気は原則として挿管するまで行わない．そのため麻酔薬投与前に十分な酸素化を行う．
    • 意識消失後は挿管が完了するまでの間，母指と示指で輪状軟骨を圧迫し，食道を閉鎖して逆流を防止する（Sellick操作）．
  • 意識下挿管（awake intubation）：
