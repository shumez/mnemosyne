急性硬膜下血腫 慢性硬膜下血腫
原因
  • 出血源は大脳皮質の血管や架橋静脈の損傷．
  • 脳挫傷の合併も多い．
  • 軽微な頭部外傷後で，出血源は不明．
  • 認知症の影響もあってか本人に外傷の記憶がないことも多い．
症状
  • 症状が重篤である．
  • 意識清明期を欠く意識障害やけいれん・片麻痺などの脳の局所症状．
  • 呼吸障害や圧脈などの頭蓋内圧亢進症状を初期から伴っていることが多い．
  • 脳の圧迫に比較して症状が軽微である．
  • 意識障害はないか，あっても軽度．
  • 軽い頭痛，上下肢の軽度運動麻痺．
  • 認知症症状が前面に出てくることも多い．
CT
  • 硬膜下腔に三日月状の高吸収領域．
  • 三日月状の低吸収領域～等吸収領域，ときに高吸収領域．
  • 血腫腔内にniveauを形成したり，低吸収と高吸収が混在することがある．
治療
  • 全身麻酔下で緊急開頭血腫除去．
  • 外減圧を追加する場合がある．
  • 脳挫傷・脳浮腫を伴っているので脳圧降下薬投与やバルビタール療法などを併用．
  • 局所麻酔下に穿頭術を行い，血腫を除去・洗浄，ドレナージ．
予後
  • 概して予後は良くない．
  • 神経脱落症状やけいれん発作を残しやすい．
  • 予後は良好．
