**Diagnosis**

[![][img_FD]][img_FD]

* Rome IV
	* ≥ 1
		1. 食後のもられ感
		2. 早期飽満感
		3. 心窩部痛
		4. 心窩部灼熱感
		* 1,3: PDS (食後愁訴症候群)
		* 2,4: EPS (心窩部痛症候群)
	* –6moから起こる, ≥3mo症状が継続
	* 器質的疾患が上部内視鏡検査で確認されない

**Treatment**

* 初期
	* PPI / H~2~RA
	* [acotimide]
* 二次
	* 抗うつ
	* 抗不安
	* 漢方

## Gastric Cancer

**Etiology**

* EB virus

**Treatment**

* CCT
	* Primary
		* HER2(–)
			* [S-1] ( / [capecitabine]) + [CDDP] (/ [oxalliplatin])
		* HER2(+)
			* [capecitabine] (/ [S-1]) + [CDDP] (/ [oxalliplatin]) + [trastuzumab]
	* Secondary
		* [capexitabin]
		* [irinotecan]
		* [ramucirumab]
	* Postope
