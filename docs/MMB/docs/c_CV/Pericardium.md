# Pericardium

## Intro

STの変化

- ST上昇（心筋の全層性虚血）
    - 急性心筋梗塞
    - 異型狭心症
    - 急性心膜炎
    - 心室瘤
- ST下降（心内膜下の虚血）
    - 労作性狭心症
    - 低K血症
    - 心内膜下梗塞

ST上昇の鑑別
