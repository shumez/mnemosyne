<h6 id='thyrotoxicosis-cls'>Classification</h6>

- ↑甲状腺機能
	- 甲状腺刺激物質: TRAb / HCG
		- Graves
		- 妊娠甲状腺中毒症
		- 絨毛性癌
- ↑TSH
	- TSHoma
- ↑自律活動
	- Plummer
	- ...
- →甲状腺機能
	- 濾胞破壊
		- 亜急性甲状腺炎
		- 無痛性甲状腺炎
	- 人為的
	- Drug-induced
	- 異所性甲状腺組織
		- 卵巣甲状腺腫
