## Leprosy / Hansen 

<!-- **Definition** -->
**Etiology**

- 起因菌  
    らい菌Mycobacterium leprae（抗酸菌）
- 感染経路  
    皮膚創傷，鼻咽頭粘膜  
    感染には乳幼児期からの濃厚な接触が必要（成人にはほとんど感染のリスクはない）らい予防法は平成7年に廃止
