Oesophagus¶

Achalasia¶

Definition
  • Auerbach plexusの変性, 消失
  • 弛緩不全
Epidemiology
  • どの年代でも, peak 20-40s
Classification

  • St: Straight type (直線型)
  • Sg: Sigmoid type (シグモイド型)
  • aSg: Advanced Sigmoid type

(Kato, Ryo (2019))

Sign and Symptom
  • 嚥下障害
  • 食道残渣の口腔内逆流
  • 胸痛, 胸焼け
Examination

  • 食道造影
    • 狭窄より口側の辺縁平滑な拡張
    • 直線型 → sigmoid型
  • 食道内圧測定
    • 食道蠕動波の消失
    • LES圧の上昇
    • LESの嚥下性弛緩性反射消失
    • 異常収縮波の出現 (同期性収縮)
  • endoscopy
Treatment
  • 薬物
    • CCB (Niphedipine) PO
    • 亜硝酸薬 SL
  • 内視鏡的バルーン拡張術
  • Surg
    • Heller筋層切開術 + 噴門形成術 (Dor手術)
Prognosis
  • 食道癌の合併率高い (5%)
Appendix
  • POEM (内視鏡的食道筋切開術)
  • ボツリヌス毒素

Esophagitis / Esophageal Ulcer¶

Etiology
  • 逆流性
    • GERD
    • 胃切除後のGERD
  • 感染性
    • カンジダ
    • HSV
    • CMV
    • TB
  • 薬剤性
    • NSAIDs
    • ステロイド
  • 腐食性
  • アレルギー性
    • EoE
  • その他
    • Crohn
    • Behçet
    • SSc
    • 天疱瘡
    • 食道静脈瘤内視鏡治療後

GERD¶

Etiology
  • LES機能不全
    • TLESR 一過性のLES弛緩
      • 高脂肪摂取
    • LES圧低下
      • 食道裂孔ヘルニア
      • SSc
      • 薬剤 (CCB, 亜硝酸薬)
      • ↑腹圧 (前屈位, 肥満, 妊娠, 亀背)
      • 噴門機能不全
        • 胃切除
        • Heller ope
  • 胃酸分泌過多
    • H.pylori
    • Zollinger-Ellison synd
  • 食道クリアランス不全
    • 食道蠕動運動
      • SSc
      • DM
      • 加齢
    • 唾液分泌障害
      • Sjögren
  • その他
    • 重症心身障害児
    • 食道閉鎖
    • 先天性横隔膜ヘルニア術後
Classification
  • びらん性GERD
  • 非びらん性GERD NERD
    • 内視鏡で食道粘膜にびらん,潰瘍などの粘膜傷害(–)
Sign and Symptom
  • 胸焼け, 呑酸
    • 前屈位
    • 食後
    • 仰臥位
  • NCCP (non-cardiac chest pain) 非心臓性胸痛
  • 咳嗽
  • 咽頭症状
  • 歯牙酸蝕 (上顎内側)
Association
  • Barrett食道
Examination
  • 内視鏡
    • mucosal break 粘膜傷害
  • 24時間pHモニタリング
  • 食道内圧測定
    • ↓LES静止圧

Revised Los Angeles classification

Treatment
  • 生活指導
    • 肥満是正
    • Fowler位
  • 薬物
    • PPI
    • H2RA
    • 消化管運動促進 (mosapride)
    • 六君子湯
  • surg
    • 逆流防止術
      • Nissen
      • Toupet

Barret esophagus¶

Definition
  • 下部食道が扁平上皮 → 円柱上皮 に置換

EoE¶

好酸球性食道炎

Etiology
  • 食物抗原に刺激 ⇒ 消化管粘膜のアレルギー反応
Examination
  • 内視鏡
    • 白班
    • 縦走溝
    • 気管様狭窄
  • Biopsy
    • eos浸潤 ≥ 15/HPF

Esophageal Candidiasis¶

Etiology
  • 原因菌
    • Candida albicans

  • 細胞性免疫低下

  • 全身的要因

    • 悪性腫瘍
    • 膠原病
    • HIV
    • DM
  • 局所的要因
    • 食道炎
    • 食物の停滞, 異物の傷害
    • 吸入ステロイド
    • PPI
Examination
  • 消化管造影
    • 敷石状陰影
  • 内視鏡
    • mm 小隆起性の白色付着物 (白苔)　びまん性, 縦列状
Diagnosis
  • glycogenic acanthosis
  • HSV
  • CMV

MWS¶

Mallory-Weiss syndrome

Definition
  • 腹腔内圧の上昇 (嘔吐)
Treatment
  • 保存的治療 → 自然止血
  • クリッピング
    • 大量出血のときは
Appendix
MWS SRE
EGJ付近 食道下部左側
S/S 出血 (++) (±)
疼痛 (–) (+)
呼吸困難 (–) (+)
ショック (–) (+)
Ex XR nl
  • 縦隔気腫, 胸水, PTX
内視鏡 EGJ付近に裂創 C/I
治療
  • 保存的治療
  • 内視鏡的止血
  • 緊急手術

SRE¶

Spontaneous rupture of esophagus 特発性食道破裂 (Boehaave)

Epidemiology
  • 食道下部 左側壁
Examination
  • ↓肺呼吸音
  • ↓声音振盪
    ∵ ↑胸水
Treatment
  • 緊急手術

  • 横紋筋腫 中下部
  • GIST
  • 乳頭腫
  • 血管腫

Esophageal Cancer¶

Epidemiology
  1. Mt (胸部中部)
  2. Lt (胸部下部)
  3. Ut (胸部上部)
Classification
  • Patho
    • Squamous (90%) ≫ Adenoma (欧米では半数がadenoma)
    • adenoma:
      • 発生母地
        • 食道固有腺
        • 食道噴門腺
        • 異所性胃粘膜
        • Barrett粘膜
      • 胸部下部食道 (Lt)
  • 病型
    • 0型: 表在型
      • 0-I: 表在隆起型
        • 0-Ip: 有茎性
        • 0-Is: 無茎性 (広基性)
      • 0-II: 表面型
        • 0-IIa: 表面隆起型
        • 0-IIb: 表面平坦型
        • 0-IIc: 表面陥凹型
      • 0-III: 表在陥凹型
    • 1型: 隆起型
    • 2型: 潰瘍限局型
    • 3型: 潰瘍浸潤型
    • 4型: びまん浸潤型
    • 5型: 分類不能型
      • 5a: 未治療
      • 5b: 治療後
  • TNM
    • 表在癌
      • ⟶ 粘膜下層(SM)
    • 進行形
      • 固有筋層 (MP) ⟶
Sign and Symptom

  • 直接浸潤

    • 下行大動脈
    • 気管
    • Lt 気管支
    • LA
    • 肺V
    • 肺動脈は近接していない

  • Metastasis

    • リンパ行性
    • 胸部, 縦隔, 頸部, 腹部リンパ節
    • リンパ節, 粘膜下リンパ管を介して, 壁内転移(intramural metastasis 飛石転移)
Examination
  • Marker
    • SCC
    • CYFRA
    • CEA
  • 食道造影
    • C/I
      • 消化管穿孔
      • 食道気管瘻形成
  • 内視鏡
    • ルゴール 不染領域
Treatment
  • 粘膜癌
    • EMR
    • ESD
    • 放射線療法 (60Co, lineac)

術後合併症

  • 早期
    • 肺合併症
      • 無気肺
      • 肺炎
      • 肺水腫
    • 循環器系
      • 術後不整脈
      • 頻脈
      • ショック
    • 縫合不全
    • 反回神経麻痺
    • 胸管損傷 ⇒ 乳び胸
    • 肝障害
    • MRSA肺炎・腸炎
  • 晩期 +2-3w
    • 吻合部狭窄
    • GERD (胸腔内吻合)
Appendix
  • JCOG1109

Esophageal Varices¶

Etiology
  • ↑門脈圧
Examination
  • RC (Red Color sign) ⟹ IND 内視鏡的治療
Treatment

  • 内視鏡
    1. EIS 硬化療法
      C/I :
      • 高度肝障害
      • Plt < 2万
      • DIC
      • 腎機能低下
    2. EVL 結紮療法
  • IVR
    • B-RTO
      • 左腎静脈に
      • IND Lg-cf (噴門部-弯窪部)/Lg-f(弯窪部) (下部)
  • surg
    • 食道離断術
  • 薬物
    • βB
    • 硝酸薬

出血時

  • Stengstaken-Blakemore tube (S-B tube)
Appendix
  • Eck瘻症候群 (術後脳症) (門脈-IVC吻合術)

Esophageal Diverticulum¶

Classification

圧出性憩室
(仮性)
  • 食道内圧の亢進
  • 筋層(–)
Zenker憩室
  • 咽頭食道移行部の食道筋層 抵抗減弱部 (Laimer三角)
  • 高齢 M
  • 手術適応(+)
横隔膜上憩室
  • 横隔膜上 抵抗減弱部位
牽引性憩室
(真性)
  • 外部との癒着
  • 気管分岐部
  • 結核性リンパ節炎の瘢痕治癒
Rokitansky憩室
  • 気管分岐部
  • 手術適応(–)
