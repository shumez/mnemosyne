Oesophagus¶

Achalasia¶

Definition
  • Auerbach plexusの変性, 消失
  • 弛緩不全
Epidemiology
  • どの年代でも, peak 20-40s
Classification

  • St: Straight type (直線型)
  • Sg: Sigmoid type (シグモイド型)
  • aSg: Advanced Sigmoid type

(Kato, Ryo (2019))

Sign and Symptom
  • 嚥下障害
  • 食道残渣の口腔内逆流
  • 胸痛, 胸焼け
Examination

  • 食道造影
    • 狭窄より口側の辺縁平滑な拡張
    • 直線型 → sigmoid型
  • 食道内圧測定
    • 食道蠕動波の消失
    • LES圧の上昇
    • LESの嚥下性弛緩性反射消失
    • 異常収縮波の出現 (同期性収縮)
  • endoscopy
Treatment
  • 薬物
    • CCB (Niphedipine) PO
    • 亜硝酸薬 SL
  • 内視鏡的バルーン拡張術
  • Surg
    • Heller筋層切開術 + 噴門形成術 (Dor手術)
Prognosis
  • 食道癌の合併率高い (5%)
Appendix
  • POEM (内視鏡的食道筋切開術)
  • ボツリヌス毒素

GERD¶

Etiology
  • LES機能不全
    • TLESR 一過性のLES弛緩
      • 高脂肪摂取
    • LES圧低下
      • 食道裂孔ヘルニア
      • SSc
      • 薬剤 (CCB, 亜硝酸薬)
      • ↑腹圧 (前屈位, 肥満, 妊娠, 亀背)
      • 噴門機能不全
        • 胃切除
        • Heller ope
  • 胃酸分泌過多
    • H.pylori
    • Zollinger-Ellison synd
  • 食道クリアランス不全
    • 食道蠕動運動
      • SSc
      • DM
      • 加齢
    • 唾液分泌障害
      • Sjögren
  • その他
    • 重症心身障害児
    • 食道閉鎖
    • 先天性横隔膜ヘルニア術後
Classification
  • びらん性GERD
  • 非びらん性GERD NERD
    • 内視鏡で食道粘膜にびらん,潰瘍などの粘膜傷害(–)
Sign and Symptom
  • 胸焼け, 呑酸
    • 前屈位
    • 食後
    • 仰臥位
  • NCCP (non-cardiac chest pain) 非心臓性胸痛
  • 咳嗽
  • 咽頭症状
  • 歯牙酸蝕 (上顎内側)
Association
  • Barrett食道
Examination
  • 内視鏡
    • mucosal break 粘膜傷害
  • 24時間pHモニタリング
  • 食道内圧測定
    • ↓LES静止圧
Treatment
  • 生活指導
    • 肥満是正
    • Fowler位
  • 薬物
    • PPI
    • H2RA
    • 消化管運動促進 (mosapride)
    • 六君子湯
  • surg
    • 逆流防止術
      • Nissen
      • Toupet
