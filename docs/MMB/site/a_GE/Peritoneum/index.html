Femoral Hernia¶

  • 嵌頓しやすい
Diagnosis

大腿 / 閉鎖孔ヘルニア

大腿 閉鎖孔
Epid 中高年以降 F 高齢 F
好発側 右（80%が片側性） (–)
頻度 鼠径ヘルニアの次に多い まれ
背景
  • 女性は大腿輪が大きい
  • 分娩による大腿輪の脆弱化
  • 骨盤が大きい ⇒ 閉鎖孔が大きい
  • 老化/多産 ⇒ 骨盤支持組織の脆弱化
  • やせ

  • ヘルニア嵌頓で発症
  • 絞扼性イレウス
  • イレウス
  • Howship-Romberg sign
嵌頓 起こしやすい 起こしやすい
診断
  • 鼠径ヘルニアと間違われることがある
  • 体表からは触れにくいためCT，USが有用
治療
  • 全例手術適応
    ⟹ メッシュによる大腿輪の閉鎖（大腿輪を縫縮）
  • 全例手術適応
