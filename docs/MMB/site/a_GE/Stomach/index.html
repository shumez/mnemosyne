FD¶

機能性ディスペプシア Functional dyspepsia

Definition
  • FGID
    • GERD
    • FD
    • IBS
Diagnosis

  • Rome IV
    • ≥ 1 (1,3: PDS (食後愁訴症候群), 2,4: EPS (心窩部痛症候群))
      1. 食後のもられ感
      2. 早期飽満感
      3. 心窩部痛
      4. 心窩部灼熱感
    • –6moから起こる, ≥3mo症状が継続
    • 器質的疾患が上部内視鏡検査で確認されない
Treatment
  • 初期
    • PPI / H2RA
    • acotimide
  • 二次
    • 抗うつ
    • 抗不安
    • 漢方
Appendix
  • HP関連機能性ディスペプシア:
    HP除菌後 6-12moでSxが消失,改善する
