Cholangio¶

Choleithiasis¶

胆石症 Gallstone

Epidemiology
  • 5Fs:
    • Female: 女性
    • Forty: ≥ 40yo
    • Fatty: 肥満
    • Fertile: 多産
    • Fair: 白人
Classification

胆石 組成による分類

  • 純コレステロール :
    • 放射状
    • 白色 - 黄白色
  • ビリルビンCa :
    • 層状
    • 茶褐色
  • 混合石 / 混成石
Sign and Symptom
  • Rt背部 - Rt肩への放散痛
Complication
  • AOSC
Diagnosis

結石

コレステロール系 ビリルビン系
発生パターン 単発 多発
好発部位
  • 胆囊
  • 肝内
  • 胆管
  • 胆囊壁内
合併しやすい
基礎疾患
  • 糖尿病
  • 肥満
  • 高脂血症
  • 溶血性貧血
  • 胃全摘術後
  • 肝硬変
超音波検査 音響陰影強い 音響陰影弱い
単純X線 石灰を含まず
結石を描出できないものも多い．		 石灰を含む結石が多く
高吸収域として描出されやすい．
Treatment
  • 胆嚢
    1. Lap-C #1
      C/I : AOSC, 胆道消化管瘻, 癌の合併, 肝硬変 etc
    2. 経口胆石溶解薬
      • IND :
        • ≤ 1cm
        • XR(-)
        • コレステロール結石
      • 胆嚢機能nl
    3. ESWL
      IND : 単発, < 20cm, 石灰化(–)純コレステロール, 胆嚢機能nl
  • 総胆管

Mirizzi syndrome¶

Definition

Etiology
  • 胆石の胆囊頸部嵌頓
  • 炎症の波及
    ⇒ 総肝管 / 総胆管の狭窄
Treatment
  • 経皮経肝胆道ドレナージ (PTBD)

Postcholecystectomy Syndrome¶

胆嚢摘除後症候群

Definition
  • 胆摘後に Rt hypochondriac 季肋部 痛を繰り返す
Etiology
  • 胆嚢頸部遺残
  • 胆石遺残
  • 瘢痕化

による総肝管の狭窄

Billary Tract Infection¶

胆道感染症

Etiology
  • 大腸菌 E. coli
  • Klebsiela
  • クロストリジウム

  • 嫌気性菌

  • 上行感染

Acute Cholangitis¶

急性胆菅炎

Treatment
  • 保存的
  • 胆道ドレナージ
    1. 内視鏡的胆道ドレナージ (EBD, ENBD)
      内視鏡的乳頭括約筋切開術 EST
    2. 経皮経肝胆道ドレナージ PTBD

AOSC¶

重症急性胆管炎

Acute Obstructive Suppurative Cholangitis 急性閉塞性化膿性胆管炎

Etiology
  • ↑胆道内圧
    ⇒ endotoxin を含む胆汁が血管内に逆流
    ⇒ 敗血症
    ⇒ DIC
    ⇒ MODS
Sign and Symptom
  • Raynolds 5徴 :
    • Charcot 3
      1. 黄疸
      2. 発熱
      3. 腹痛
    • 意識障害
    • ショック
Treatment
  • 輸液
  • ABx
  • 呼吸循環管理
  • drainage

C/I:

  • Lap-C
  • PTBD

  • U/S
    • コメット（彗星）様エコー
Treatment
  • f/u
  • s/o 癌 ⟹ 胆嚢摘出術

Billary Tract Cancer¶

胆道癌

Etiology
  • 胆嚢癌:
    • 膵胆管合流異常 (胆道非拡張型)
    • 陶磁器様胆嚢
  • 胆管癌
    • 膵胆管合流異常 (胆道拡張型)
    • PSC
    • 肝内結石
  • 乳頭部癌: (–)
Classification

Sign and Symptom
  • Courvoisier sign
    • 無痛性の胆嚢腫大
    • 3管合流部以下の胆管閉塞
    • e.g.
      • 膵頭部癌
      • 遠位胆管癌
