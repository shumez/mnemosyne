FH / ALF¶

Fulminant Hepatitis 劇症肝炎 / Acute Liver Failure 急性肝不全

Definition
  • 初発症状出現から ≤ 8w (56d)に, PT ≤ 40% / INR ≥ 1.5
Etiology
  1. HBV
Sign and Symptom
  • 腎不全 / 出血傾向 / 腹水
  • 肝萎縮 / 肝濁音界縮小
  • 意識障害 (↑頭蓋内圧)
Examination
  • PT ≤ 40%
  • L/D
    • ↓Chol / ↓Alb / ↓ChE
    • ↑Bil / ↑HN3 / ↑AAA
Prognosis
  • 亜急性型 予後不良
    • 急性: 50%
    • 亜急性: 10%
