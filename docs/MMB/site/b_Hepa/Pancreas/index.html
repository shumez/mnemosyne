Pancreas¶

AP¶

急性膵炎

Etiology
  1. アルコール 35％
    • 男性に多い
  2. 胆石症 30％
    • 女性に多い
    • 乳頭部への嵌頓による．
  3. 特発性 (原因不明) 20％
    • 女性に多い
  4. 医原性
    • 診断的ERCP
    • 内視鏡的乳頭処置
    • 手術
Complication
  • MODS
  • 膵仮性嚢胞
Examination
  • L/D
    • ↑AMY 遷延
      ⟹ 仮性膵嚢胞の合併
      ↑ACCR (AMY Cr Clearance Ratio)
    • ↑BS ∴ insulin(–)
  • AXR: Colon cut-off sign
    腸管麻痺
C-TOP
C Ca
Chol
T TProtein
O O2
P Plt

造影CT Grade：
炎症の膵外進展度と造影不良域から重症度を判定する．
※造影不良域：膵を膵頭部，膵体部，膵尾部の3つの区域に分けて判定する．

Treatment

  • 基本的
    • 輸液
    • 呼吸/循環管理
    • タンパク分解酵素阻害薬
    • NG Tube
      腸管閉塞, N/Vを伴う場合
  • 重症例
    • 膵動注療法 ABx
  • 感染性膵壊死 / WON
  • 胆石性膵炎
Appendix

重症度判定

  • 予後因子：
    以下の項目を各1点とする．
    1. Base Excess（BE）≦－3mEq/Lまたはショック（収縮期血圧≦80mmHg）
    2. PaO2≦60mmHgまたは呼吸不全（人工呼吸を必要とする）
    3. BUN≧40mg/dL（またはCr≧2.0mg/dL） または乏尿（輸液後も一日尿量≦400mL）
    4. LDH基準値上限2倍以上
    5. 血小板数≦10万/mm3
    6. 低Ca血症（総Ca値≦7.5mg/dL）
    7. CRP≧15mg/dL
    8. SIRS診断基準陽性項目数≧3
    9. 年齢70歳以上

SIRS（全身性炎症反応症候群）診断基準項目

  • 体温＞38℃または＜36℃
  • 脈拍＞90回/分
  • 呼吸数＞20回/分またはPaCO2＜32mmHg
  • 白血球数＞12,000/mm3または＜4,000/mm3または10％超の幼若球出現

膵炎の合併症

壊死(–) 壊死(+)
≤ 4w 急性膵周囲液体貯留 急性壊死性貯留
> 4w 膵仮性嚢胞 被包化壊死 (WON)

CP¶

慢性膵炎

Sign and Symptom
  • DM
  • 局所門脈圧亢進症
    ∵ CP →脾Vに及んで ⇒ 左側PoV 圧↑
Examination

  • BT-PABA ≤ 70%
  • MRCP / ERCP
    • 狭窄 / 拡張
    • 数珠状拡張
  • 石灰化
Treatment
  • 代償期
  • 移行期
    • 生活指導
      • 低脂肪食
        ∵ 脂肪　⇒ ↑膵液
    • 薬物
      • PPI / H2B
        ∵ 胃酸 ⇒ ↑膵液

      • ∵ 胆汁酸 ⇒ ↑膵液

  • 非代償期
  • 合併症

AIP¶

自己免疫性膵炎

Complication
  • IgG4
    • 涙腺 / 唾液腺炎
    • PSC
Examination

  • ↑IgG4
  • MRCP
    • 主膵管の狭細
Treatment
  • 内視鏡的ドレナージ

Pancreas Cancer¶

Epidemiology
  • 膵頭部 60%
Examination
  • L/D
    • CA19-9(+)
  • Contrast-enhanced
    • 膵管拡張
  • MRCP
    • ERCPでは見えない 主膵管閉塞部上流を描出
  • ERCP
    • 主膵管 閉塞/狭窄
Treatment

  • Resectable
    • PD
    • 膵体尾部/脾合併切除
  • Borderline resectable
  • Unresectable
Appendix

主膵管/総胆管 壁不整/途絶

  1. 膵頭部癌
  2. 下部胆管癌
  3. 十二指腸乳頭部癌
