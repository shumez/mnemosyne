  • HC
    • 治療終了から6mo

IFN¶

Interferon

Classification
IFN-α IFN-β
α α 2a, α 2b β
産生細胞 WBC Fibroblast
投与 筋注, 皮下注 IV
製剤 天然型 遺伝子組み換え 天然型
Pharmacology

Menghao Huang, 2014

Indication
  • HB 慢性肝炎
    • HBeAg の陰性化
    • HBV-DNAの低下
    • ALT 持続的正常化
  • HC 慢性肝炎
    • HCV 陰性化
    • AST/ALT 正常化
    • 発がんの予防
Adverse Effect
  • 発熱
  • 間質性肺炎
  • 抑うつ
  • ↓WBC, ↓Plt
Contraindication
  • 自己免疫性肝炎
  • 小柴胡湯 投与中
  • IFN 過敏症
