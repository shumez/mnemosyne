WPW¶

Woff-Parkinson-White Syndrome

Diagnosis

WPW ⇒ PSVT

  • RR reg
  • narrow QRS

RR間隔が規則的な頻拍．房室結節を順行伝導し副伝導路を逆行伝導するためQRS幅は狭い

WPW ⇒ pAF

  • RR irreg
  • wide QRS

RR間隔が不規則な頻拍．副伝導路を介して順行伝導するためδ波を伴った幅の広いQRS波形になる

Treatment
  • PSVT
    • ATP
    • CCB
  • pAF
    • AAD Ia
      • Disoperamide
      • Procainamide
        副伝導路の伝導に対して抑制的
